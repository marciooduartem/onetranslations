# One Translations — Conversor PDF → Word

## Como fazer o deploy no Railway

### 1. Criar conta no GitHub
- Acesse github.com e crie uma conta gratuita

### 2. Criar repositório
- Clique em "New repository"
- Nome: `onetranslations`
- Clique em "Create repository"
- Faça upload de todos os arquivos desta pasta

### 3. Deploy no Railway
- Acesse railway.app
- Clique em "Start a New Project"
- Selecione "Deploy from GitHub repo"
- Conecte sua conta GitHub
- Selecione o repositório `onetranslations`
- Clique em "Deploy Now"

### 4. Pronto!
O Railway vai instalar tudo automaticamente e gerar um link como:
`https://onetranslations.up.railway.app`

## Funcionalidades
- Formatação Simples: PDF digital (direto) ou escaneado (via IA)
- Tradução Juramentada: sempre via IA com substituições padronizadas
- Modo CETRA: regras específicas para documentos Cetra/Chery
